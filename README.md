# hagaki
addressee printer for Japanese postcard

- はがき宛名印刷を行います

- Microsoft Wordのない環境でも印刷可能です
    - 住所データの読み込みにはcsvまたはopenpyxlを使用
    - PDF生成はfpdf2のFlexTemplateを使用

- 以下のフォントをテストに使用しています
    - Windows 11同梱のフォント
        - BIZ UD明朝
        - UDデジタル教科書体
        - 游明朝
    - Office同梱のフォント
        - HG正楷書体PRO
        - HG教科書体
    - フリー楷書体フォント
        - 英推楷書 https://www.ac-font.com/jp/detail_jb_007.php
